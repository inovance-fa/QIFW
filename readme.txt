Introduction
------------
This application is a Windows platform packaging tool developed based on 
qt installer framework. Implement component packaging through command line 
invocation instructions.

Acknowledgments
---------------
This application is developed based on qt installer framework version 4.4.0, 
and the Qt source code is modified in accordance with the GPL V3 protocol.
For further information, see the project home page:
    https://download.qt.io/official_releases/qt-installer-framework/4.4.0/

Compilation Dependencies
---------------
1.installer-framework-opensource-src-4.4
2.Qt5.15.2-stati
